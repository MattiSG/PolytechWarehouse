Aide
  • Composer un groupe de rendu
    Permet de composer un groupe de rendu pour livrer le travail (uniquement si vous n'avez pas de groupe, ou si vous en avez un et qu'il est encore possible d'ajouter des membres). Si une date limite de composition des groupes existe, vous ne pouvez plus composer votre groupe au-delà de cette date. Sinon, vous avez jusqu'à la fin du rendu le composer.
  • Voir les informations sur le groupe de rendu
    Permet de visualiser les informations sur votre groupe de rendu, et de le quitter (uniquement si vous faîtes partie d'un groupe).
  • Téléchargement
    Permet de télécharger l'archive contenant tous les dernieres versions de chacun des fichiers à rendre (uniquement si vous avez réalisé une livraison auparavant).
  • Livraison du fichier X
    Permet d'envoyer un fichier pour valider son travail. Le fichier envoyé doit respecter le format spécifié. Vous pouvez re-livrer indépendement chacun des fichiers à rendre.
