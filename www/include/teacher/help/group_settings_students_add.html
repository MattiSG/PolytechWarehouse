Aide

Utilisez les onglets de navigation pour choisir de configurer le nom du groupe, ajouter ou supprimer des étudiants dans le groupe, créer de nouveaux étudiants ou éditer le profil de ceux déjà existants.

  • Ajout d'étudiants
    Permet d'ajouter des étudiants dans le groupe. Pour ajouter un étudiant, parcourez l'index en fonction de son nom puis cliquez sur la ligne lui correpondant pour l'ajouter. Pour valider vos choix, cliquez sur le bouton "Ajouter +".
