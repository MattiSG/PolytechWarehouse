Aide
  • Configuration du fichier X
    Saisissez le nom du fichier que vous souhaitez être livré ainsi que son type. Les fichiers envoyés par les étudiants seront renommés par le nom spécifié. Le choix du type de fichier permet 2 types de séléction. Vous pouvez choisir un type général par exemple comme "document", "image", ou "archive", qui vont respectivement accepter tous type de documents, d'images ou d'archives. Vous pouvez aussi choisir un type précis par exemple comme "pdf", "png", ou "zip", qui vont uniquement accepter les fichiers portant respectivement des fichiers d'extension ".pdf", ".png" ou ".zip".
  • Ajouter/Supprimer un fichier
    Permet d'ajouter/supprimer une fichier à rendre par les étudiants.
