Aide

Vous pouvez ici créer des groupes en fonction de la hierarchie de l'école. Par exemple, à partir de la promotion SI4, vous pouvez créer les sous-groupes Gr1 et Gr2 puis respectivement les sous-groupes Gr1a, Gr1a et Gr2a, Gr2b.

  • Nom du groupe
    Saisissez ici le nom du groupe que vous voulez créer. Au sein d'un groupe, vous ne pouvez pas créer 2 sous groupes qui portent le même nom.
  • Groupe parent
    Indiquer ici le groupe dans lequel vous voulez créer le sous-groupe.
