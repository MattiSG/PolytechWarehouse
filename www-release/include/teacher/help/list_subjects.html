Aide

Les matières disponibles sont listées sur cette page. Vous pouvez afficher toutes les matières disponibles ou seulement celles que avez précédement créées en cliquant sur le lien en bas du tableau. Pour consulter les travaux d'une matière, cliquez sur son nom.

  • Créer une nouvelle matière
    Permet de créer une matière. Utilisez cette fonctionalité lorsque la matière que vous enseignez n'est pas disponible parmi les matières déjà existantes.
  • Configurer
    Permet de modifier le nom d'une matière, d'ajouter ou supprimer les enseignants qui en sont responsables.
  • Supprimer
    Permet de supprimer une matière uniquement si vous faîtes partie des enseignants responsables.
