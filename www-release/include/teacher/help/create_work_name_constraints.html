Aide
  • Nom du travail
    Saisissez ici le nom du travail que vous voulez créer.
  • Membre minimum
    Saisissez ici le nombre minimum d'étudiants autorisés dans un groupe pour réaliser ce travail.
  • Membre maximum
    Saisissez ici le nombre maximum d'étudiants autorisés dans un groupe pour réaliser ce travail.
  • Site web du sujet
    Saisissez ici le lien du site web contenant le sujet du travail. (Non obligatoire).
  • Charge de travail
    Saisissez ici une appoximation de la quantité d'heures par étudiant que nécessite la réalisation de ce travail. (non obligatoire).
  • Tolérance
    Saisissez ici le nombre de jour de retard toléré. La tolérance vous permet d'accepter des travaux rendus en retard et d'en être informé (non obligatoire).
  • Taille
    Saisissez ici la taille maximale des fichiers livrés par les étudiants. (non obligatoire).
  • Rendu express
    Choisissez ici le type de travail. Dans le cas d'un rendu "Normal", les étudiants disposent d'une periode de composition des groupes. Pendant cette période, ils peuvent constituer des groupes de rendu, les modifier, les quitter, mais ne peuvent pas effectuer de livraison. Au terme de cette période, les groupes sont figés et seuls ceux respectant les contraintes de groupes peuvent effectuer des livraisons jusqu'à la date limite de rendu. Dans le cas d'un rendu "express", les étudiants ne disposent pas de la période de composition des groupes. Pour effectuer des livraisons, ils doivent constituer un groupe de rendu qu'ils ne pourront plus quitter ensuite. Dès que le groupe de rendu est formé et respecte les contraintes de groupes, les étudiants peuvent effectuer des livraisons jusqu'à la date limite de rendu. Dans les 2 cas, vous pouvez intervenir dans la composition des groupes de rendu, c'est-a-dire en créer directement vous-même et supprimer ceux créés par les étudiants.
