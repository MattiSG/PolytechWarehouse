Aide
  • Composition des groupes de rendu
    Permet de composer les groupes de rendu avec les étudiants sans groupe, uniquement si la date limite de rendu n'est pas dépassée.
  • Groupe de rendu existant - Suppression
    Permet de supprimer un groupe de rendu. La suppression d'un groupe entraîne aussi la suppression de tous les fichiers livrés par les étudiants.
